What is CSS?

Cascading Style Sheets (CSS) is used to fomat the layout of a webpage.
With CSS, you can control the color, font, the size of text, the spacing between elements, how elements a positioned and laid out, what background imgages or background colors are to be used, different displays for different devices and screen sizes, and much more!
Tip: The word cascading means that a style applied to a parent, elements will also apply to all children elements within the parent. So, if you set the color of the body tex to "blue", all headings, paragraph, and other text elements within the body will also got the same color (unless you specify something else)!
